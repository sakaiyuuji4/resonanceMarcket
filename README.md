#### 介绍
此为Yunzai-Bot云崽QQ机器人的插件，需要依赖云崽机器人来实现功能

### 插件功能
通过获取实时跑商信息并返回最佳跑商路线

### 使用方法
将[resonanceBase.json](resonanceBase.json)放入Yunzai/data文件夹中
将[雷索纳斯时价表查询.js]放进plugin/example文件夹即可。
在群聊或私聊中发送 #时价表查询
